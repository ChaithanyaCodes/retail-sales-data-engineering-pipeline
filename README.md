# Retail Sales Data Engineering Pipeline & Analytics Dashboard
End-to-end Data Engineering pipeline using Python, SQL, SQLite, and Data Visualization
## Overview

This project demonstrates an end-to-end Data Engineering pipeline that ingests retail sales data, performs data transformations, loads processed data into a database, and generates analytical dashboards.

The solution simulates a real-world batch data pipeline used for business reporting and analytics.

## Project Features

### Data Ingestion
- Reads sales transaction files from CSV
- Handles structured data ingestion

### Data Transformation
- Calculates total revenue
- Converts date formats
- Performs data quality checks
- Removes duplicate records

### Data Storage
- Loads curated data into SQLite database

### Reporting
Created SQL reports for:

- Total revenue
- Category performance
- Product sales ranking
- Sales trends

### Dashboard

Created Python dashboards showing:

- Revenue KPIs
- Category sales
- Top products
- Sales trends
